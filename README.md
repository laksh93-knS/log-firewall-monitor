# Log and Firewall Monitoring Script 

## Description
This project is a simple Bash script that checks system logs and firewall status.

It performs:
- Displays recent failed login attempts
- Shows firewall status
- Lists open network ports

The script helps in basic system monitoring and security checking.

---

## Prerequisites
No additional packages are required.

You only need:
- A Linux system
- Terminal access
- Sudo privileges

---

## Installation

Navigate to the folder:
cd log-firewall-monitor

Make the script executable:
chmod +x log_firewall_check.sh

---

## Usage

Run the script:
./log_firewall_check.sh

---
